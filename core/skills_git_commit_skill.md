# git-commit

## Overview
Generates conventional commits and optional PR descriptions.

## Category
Workflow

## Auto-Invoke
Creating commit

## Scope
Git workflow only.

## Inputs
- Change summary
- Scope
- Type

## Outputs
- Commit message
- Optional PR description

## Execution Rules
- Conventional commit format
- Clear and concise
- No AI mention

## Steps
1. Classify change type
2. Generate commit message
3. Optionally generate PR description

## Validation
- Matches conventional format
- Semantic meaning

## Prohibited
- Git commands
- Branch creation

## Version
1.0.0
